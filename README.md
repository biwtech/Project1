จะได้ตัวเว็บขึ้นมา จะมีหน้า Home เอาไว้ดูหน้าเว็บไซต์มีระบบที่สามารถเขียน รีวิวได้ ละก็มี Menu สามารถเลื่อนดูเมนููที่มีได้ละก็ค้นหาเมนูที่ร้านมีได้ ละก็ About us อธิบายต่างๆว่าทำไมต้องเป็นร้านนี้คุณสมบัติต่างๆของร้านบรรยากาศของร้านมีรูปภาพบรรยากาศร้าน มี Reserve ที่สามารถจองโต๊ะได้นั้นเอง
